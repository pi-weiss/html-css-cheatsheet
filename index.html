HTML and CSS cheat sheet

HTML

Basic Tags
<html> </html> creates an HTML document
<head> </head> Sets off the title & other info that isn't displayed
<body> </body> Sets off the visible portion of the document
Tables
<table> </table> creates a table
<table> </table> creates a table
<tr> </tr> adds row to table
<td> </td> adds data to row

CSS

CSS selectors
head selects the element with the head tag
.red selects the element with the 'red' class
#nav selects the element with the 'nav' id
div.row selects all the elements with the div tag and the 'row' class
li a DOM descendant combinator. All a tags that are a child of li tag
div .row * Selects all elements that are descendant of the elements with div tag and 'row' class
